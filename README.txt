# Getting Started

#PreRequisite
You must have installed Eclipse and JDK 1.8 +
Eclipse must be configured with Cucumber eclipse plugin and also gradle plugin
Gradle needs to be installed or you can use gradle wrapper as well

#How to run
Import the project into eclipse as existing gradle project
then go to src/main/resources/features and open feature file
right click on feature file and select run as Cucumber
You will see results in consolse
